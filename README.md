VertiCut
============

["fast search in hamming space with multi-index hashing"](https://docs.google.com/viewer?url=http%3A%2F%2Fwww.cs.toronto.edu%2F~norouzi%2Fresearch%2Fpapers%2Fmulti_index_hashing.pdf)
